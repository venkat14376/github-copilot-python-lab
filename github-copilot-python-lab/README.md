# GitHub Copilot Python Lab

This project demonstrates a FastAPI application enhanced using GitHub Copilot.

## Features

- Welcome endpoint
- Checksum endpoint
- Pydantic request model
- Automated tests

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
uvicorn main:app --reload
```

## Open API Documentation

http://127.0.0.1:8000/docs

## Run Tests

```bash
pytest
```
